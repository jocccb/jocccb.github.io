（推荐）使用 yum 安装

  1. redis在第三方的源里，首先添加源，添加 epel 源
  1. 安装 redis
  1. 配置文件在 /etc/redis.conf，配置文件和上面一样的。
  2. 启动命令 systemctl start redis，或者使用指定配置文件的方式 redis-server /etc/redis.conf。
  3. 开机自启 systemctl enable redis。
  4. 使用 redis 客户端 redis-cli

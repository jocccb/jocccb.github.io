Window 安装

目前 win 10 pro 支持 typer-V

阿里云下载

在 控制面板 》 程序和功能 》 启用或关闭 Window 功能中勾选上 Typer-V 安装并重启，并且去官网下载 Docker 的安装包。

注意的是，网站被墙，需要翻墙，不然下载龟速，或者去国内的镜像网站下载
且不是 win 10 pro 或企业版，不能启动 Typer-V 只能使用虚拟机环境了，可以下载 Docker Toolbox。
